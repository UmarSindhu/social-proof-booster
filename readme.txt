## Description

Social Proof Booster helps you display engaging social proof popups on your WooCommerce website to increase customer trust and boost sales. Showcase recent customer actions, such as purchases or reviews, in real-time with customizable popups.

### Features
- Display recent customer actions (e.g., purchases).
- Fully customizable popup appearance and behavior.
- Supports WooCommerce integration.
- Mobile-friendly design.
- Easy-to-use settings panel in the WordPress admin dashboard.

### Use Cases
- Increase trust with visitors by showcasing recent purchases.
- Boost sales by creating urgency and social proof.
- Engage users with visually appealing notifications.

## Installation

1. Upload the `social-proof-booster` plugin folder to the `/wp-content/plugins/` directory or install it directly from the WordPress plugin repository.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate to the **Social Proof Booster** settings under the WordPress admin menu to customize your popup preferences.

## Frequently Asked Questions

### Does this plugin require WooCommerce?
Yes, WooCommerce must be installed and activated to use this plugin.

### Can I customize the popup design?
Yes, you can change the popup background color, position, and delay from the settings page.

### Does the plugin work on mobile devices?
Yes, the popups are responsive and mobile-friendly.
